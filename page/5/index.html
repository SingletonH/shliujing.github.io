当我作为一位面试者时，我虚心向面试官请教，并希望面试官能够认同我的个人价值。
当我作为一位面试官时，我真诚的对待每一位面试者，并希望面试者能给我以惊艳。

刘靖

关于面试，上篇博文刚说了在2345的面试经历。详见：2345面试后记。

这次是我上次找工作时整理的一篇文档。因为当时面试的公司比较多，每次都说同样的话，做同样的事。我干脆整理一份文档，把相关问题都缕清楚，节省了节省面试双方的时间，既省时又省力。

结果表明：效果并不理想。面试关键在于快速了解，虽然有文档，但因其他接触减少，相应的会减少整体信息的获取。

其中有一家做移动导航的公司，公司名我就不说了。PM看完文档5分钟后，简单问了一两个问题，流程就Over了。我也看的出来，他不在状态，不过这样的行为对于面试者连一点起码的尊重都没有做到。

常用操作

Git的通用操作，可以在本地做好版本控制后，push到Git的服务器,可以是Github、Gitlab。

Create a new repository on the command line

git init 
git add .
git commit -m "first commit"
git remote add origin https://github.com/yourId/repoName.git
git push -u origin master

Push an existing repository from the command line

git remote add origin https://github.com/yourID/repoName.git
git push -u origin master
