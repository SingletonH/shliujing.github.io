为博客添加演示地址 - 将演示与博客代码一起发布（一）

发表于 2015-09-04   |   分类于 Tech   |     |   热度

紧接上篇手动实现类似易企秀PPT那样的HTML5网站。博文中，为了增加演示地址。我试了两种方法，一种是为博客添加演示地址 - 将演示与博客代码一起发布（一）；一种是为博客添加演示地址 - 通过SVN自动化构建代码（二）。

演示与博客代码一起发布的好处在于方便，坏处也很明显：

  1. 加大博客代码的量，增加了发布时间
  2. 代码混乱，不利于长期管理
  3. 只支持HTML静态代码的演示

所以我又试了另外一种方法为博客添加演示地址 - 通过SVN自动化构建代码（二），有兴趣可以看看，不过稍微复杂一些。
