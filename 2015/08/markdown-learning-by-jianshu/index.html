简书Markdown入门简记

发表于 2015-08-14   |   分类于 Tech   |     |   热度

之前学习Markdown的时候，入门的语法只花了10来分钟。但是在工具选择，以及工具的高效使用上，花了一些时间。我用过的工具大概有几款吧，回忆过来，用过的顺序依次是简书->马克飞象->Cmd Markdown->Mou->Sublime->IDEA->Atom。最好用的还是马克飞象和Sublime。

今天要介绍的就是一款非常简洁美观的Markdown工具：简书。一款写作与阅读平台平台软件，Markdown只是写作方式的一种选择而已。不过这个平台做的很优美，在上面写文章的话，会有一种享受的感觉，写出来的样式同样也赏心悦目。

写作环境

老实说，一两年前，我刚看到简书网站时，第一反应是这网站做的真精美，是做旅游文章分享的吧；第二反应是这个网站虽然很精美，但是由于业务特性，应该做不大吧。没想到今天能做的这么出名，媲美知乎专栏。不过现在简书上鸡汤有些过多了。

Markdown语法

简书还是在Markdown基本语法的基础上，增加了一些扩展语法。这里主要记录一些基本之外的部分扩展语法，完整的只有一篇博文长度的官方入门教程，点击这里查看
。扩展的标记我没特意去记，值需要记好日常使用最多的一套快捷键即可Sublime的这套Markdown快捷键。

# 这是一级标题

## 这是二级标题

## 无序列表

- 文本1
- 文本2
- 文本3

## 有序列表

1. 文本1
2. 文本2

## 链接和图片

在 Markdown 中，插入链接不需要其他按钮，你只需要使用 [显示文本](链接地址) 这样的语法即可，例如：
[百度](baidu.com)
在 Markdown 中，插入图片不需要其他按钮，你只需要使用  ! [](图片链接地址) 这样的语法即可，例如：
![](http://ww4.sinaimg.cn/bmiddle/aa397b7fjw1dzplsgpdw5j.jpg)
注：插入图片的语法和链接的语法很像，只是前面多了一个 ！。

## 引用

在我们写作的时候经常需要引用他人的文字，这个时候引用这个格式就很有必要了，在 Markdown 中，你只需要在你希望引用的文字前面加上 > 就好了，例如：

> 一盏灯，一片昏黄；
> 千里江陵一日还；

## 粗体和斜体

Markdown的**粗体和斜体也**非常简单，用两个 * 包含一段文本就是粗体的语法，用一个 * 包含一段文本就是斜体的语法。例如：
*一盏灯*，一片昏黄；**简书**书杯淡茶
