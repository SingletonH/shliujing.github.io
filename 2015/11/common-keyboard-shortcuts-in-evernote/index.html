印象笔记在Windows和Mac下的日常使用

发表于 2015-11-23   |   分类于 Tech   |     |   热度

我之前用的都是有道云笔记，有一次有道云更新版本把我所有的笔记分类全打乱后，我终于决定转移阵地，选用印象笔记。

用了印象笔记也快有一年了，因为笔记众多，当时转移笔记也花了不少时间，由于需要同步大量笔记的缘故，我也就购买了高级会员。但是体验到今天，我的满意度并不是很高，包括较丑的UI，和一些细节的操作方便性上。不过印象笔记强大不在于表象，而是需要去感受的其笔记工具的实用性。比如网页剪切，分类标签及其搜索效率上。但是UI真是丑，而且还不能公开分享笔记，最后还有那推出的阉割版的的Markdown写作功能…
