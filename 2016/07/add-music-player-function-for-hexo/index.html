为Hexo Next主题添加音乐播放器功能（二）

发表于 2016-07-20   |   分类于 Tech   |     |   热度

刚看完港漫《封神纪》，推荐。离睡觉点还有些时间，正好更个教程。

从创建博客始，我就有要增加音乐功能的想法。相信很多个站博主都有增加音乐功能的想法。起初我是想增加H5的音乐播放器。后来在netcan666的博客个人介绍中，看到网易音乐的播放器，也很精致。接着在一篇也是next主题的博主分享的博文中看到相关教程，非常抱歉，时间太久了，我未作参考记录，忘的也差不多了。

其实原理很简单。在网易音乐的中，生成外链播放器代码，放入source\about\index.md中即可。

步骤

这里假设为关于页面增加音乐播放器功能。

建立关于页

未建立关于页面的，可以参考添加标题页面：添加「标签」页面。

获取网易云音乐功能

登录网易云音乐网页版，在歌曲详情/歌单详情，可以分别对应外链播放器代码。生成的外链可以选择样式，总共有3种，可以根据自己喜欢做调整。

生成单首歌曲代码

生成歌单代码
歌单可以是任意的歌单，推荐生成XX最喜欢的音乐歌单。可以在我的主页 - 我创建的歌单，找到自己的歌单，再选择外链生成。

在标签页添加音乐代码

复制代码到站点文件夹source\about\index.md即可，可能需要做些微调，比如摆放位置，宽高等。

我的代码

我调试了部分宽高，可提供一些参考。

总结

过程不难，我的主要时间可以花在播放器页面的样式调整上。我的主题因为加宽了，所以相关的调整比较多，原装的主题效果会好一点。

关于调整歌曲顺序，下载网易云音乐客户端，调整歌单中歌曲顺序，即可改变对应生成的外链播放器播放的音乐顺序。

关于是否显示歌曲列表。我的音乐多是轻音乐，也没啥歌词，再一个，为了让人更关注于音乐本身，没必要加歌曲列表了。实在喜欢的，留个言，我也会告知歌曲名。这也算是化繁为简吧。所以个人认为，喜欢简单的朋友就没必要加歌曲列表了，甚至是播放器框也没必要。
